The DiscordBot-Maker project utilizes a modern web development tech stack:

**Frontend:**
- **Next.js 14 (App Router):** A React framework for building full-stack web applications.
- **React:** A JavaScript library for building user interfaces.
- **TypeScript:** A superset of JavaScript that adds static typing.
- **Tailwind CSS:** A utility-first CSS framework for rapid UI development.

**Backend/Data:**
- **Firebase Firestore:** A NoSQL cloud database for storing application data.
- **GitHub OAuth:** For user authentication and authorization with GitHub.

**Integrations/Utilities:**
- **Octokit/rest:** A client for interacting with the GitHub API.
- **Zod, React Hook Form, @hookform/resolvers:** For form validation and management.
- **Lucide-React:** An icon library.
- **Clsx, Tailwind-Merge:** Utilities for managing CSS classes with Tailwind CSS.

**Deployment:**
- **Vercel:** The platform used for deploying the DiscordBot-Maker application itself.

**Generated Bot Runtime:**
- **Cloudflare Workers:** A serverless execution environment for running the generated Discord bots.
- **Hono:** A small, fast, and modern web framework for the Edge, used for the generated bots.
